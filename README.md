# Portofolio — Zoni Fatma Mulia

Ini scaffold portofolio statis (HTML/CSS/JS). Gaya: elegan-profesional, warna cream & coklat. Isi file berikut untuk menyesuaikan:

- `index.html` — konten utama (nama, tentang, pengalaman, proyek, kontak).
- `styles.css` — styling, palet warna.
- `script.js` — skrip kecil (smooth scroll, tahun otomatis).

Langkah cepat untuk melihat di lokal:

```powershell
# buka folder ini di terminal
start .\index.html
```

Deploy cepat ke GitHub Pages:

1. Inisialisasi repo: `git init`.
2. Commit semua file.
3. Push ke GitHub dan aktifkan Pages pada branch `main`.

Silakan kirim CV (teks atau file) atau kirim koreksi konten; saya akan masukkan data pengalaman Anda ke bagian `Pengalaman` dan menyesuaikan teks/warna jika diperlukan.
